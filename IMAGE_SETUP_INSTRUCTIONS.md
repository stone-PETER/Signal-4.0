# Superhero Images Setup Instructions

## Step 1: Save Your Images

Save the 4 superhero images you provided to this location:
`c:\programs\Signal-4.0\public\assets\superheroes\`

### Recommended Filenames:

1. **Iron Man helmet (red/gold)** → Save as: `ironman-helmet.jpg`
2. **Black Panther (purple)** → Save as: `black-panther.jpg`
3. **Spider-Man crouching** → Save as: `spiderman-crouch.png`
4. **Spider-Man mask (newspaper style)** → Save as: `spiderman-mask.jpg`

## Step 2: Where Images Will Appear

- **Hero Section**: Black Panther (large background)
- **About Us**: Iron Man helmet (decorative corner)
- **Workshops**: Spider-Man crouching (floating element)
- **Contact Us**: Spider-Man mask (background decoration)

Once you save the images with these filenames, the code will automatically display them!
